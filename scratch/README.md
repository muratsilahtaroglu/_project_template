# scratch/ — temporary/experimental/exploration code (rules.md §3)

Never spread into the main source tree. Subfolders: `experiments/`, `probes/`, `one_off/`, `archive/`.
**Rule:** every scratch file starts with a 1-line purpose comment. If something proves useful and should
become permanent, it's moved into the appropriate module (+architecture.md); if not, it's deleted or
moved to `archive/`. At the end of a session, no file is left unanswered for "what is this?".
